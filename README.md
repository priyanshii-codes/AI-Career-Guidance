# 🎯 AI Career Guidance System

An interactive career recommendation tool built with **Python (Tkinter GUI)** and powered by **Machine Learning (TF‑IDF + Cosine Similarity)**.  
This project helps students and professionals discover suitable career paths based on their **skills, interests, or subjects**.

---

## 🚀 Features
- 🖥️ **User-Friendly GUI** built with Tkinter.
- 🤖 **AI-Powered Recommendations** using TF‑IDF vectorization and cosine similarity.
- 📊 Suggests **Top 3 Career Matches** based on user input.
- 📂 Provides **Projects & Internships** ideas for each recommended career.
- ⚡ Lightweight, fast, and easy to run locally.
